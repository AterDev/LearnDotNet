# 搭建项目

最快的方法是使用模板快速的创建出初始项目内容，以便让我们快速进入代码开发的阶段中。这里我将使用`Ater.Dry`工具来创建新的解决方案。

## 使用Ater.dry创建项目

安装工具

```powershell
dotnet tool install -g ater.dry 
```

运行工具

```powershell
dry studio
```

创建新解决方案，这里将后台管理模块点选上即可！

打开解决方案!
